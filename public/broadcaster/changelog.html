v1.3.0
14 May 2026
Major
Overlay Studio — Drag, Resize & Rotate
  • ● Draggable overlays — Drop any image or video file onto the preview; each becomes an independently positionable overlay node. Click and drag anywhere on the node to reposition.
  • ● Corner resize handles — Four corner handles let you scale an overlay with correct aspect-ratio locking. The Scale slider in the sidebar syncs in real-time.
  • ● Rotation handle — A circular handle below the overlay lets you rotate it freely. The Rotation slider also accepts manual degree input.
  • ● Video overlays — MP4, WebM, and other browser-supported video formats can now be dropped as overlays. They loop and play automatically. Per-overlay Loop and Mute toggles available.
  • ● Overlay list panel — All active overlays appear in a sidebar list with thumbnail, type badge, eye toggle (show/hide), and delete button. Click any row to select and edit that overlay.
  • ● Opacity & per-overlay controls — Each selected overlay exposes an Opacity slider, Scale slider, Rotation slider, and video-specific Loop/Mute checkboxes.
  • ● Duplicate overlay — ⧉ Dupe button instantly creates a copy of the selected overlay offset by a few pixels.
  • ● Drag-and-drop zone — The Overlays card now features a drop zone accepting both images and videos; no file picker required.
  • ● Canvas composite — Overlays are composited onto the broadcast canvas via ctx.drawImage with translate/rotate transforms, so viewers see them live.
  • ● Legacy fixed-position overlay removed from the main Overlay card (replaced by the new system). The Flash Overlay card remains for fullscreen urgency overlays.
  • ● Changelog page — You're reading it. Accessible via the CHANGELOG button in the topbar.

v1.2.0
12 Apr 2026
Feature
Screen Capture & Soundboard
  • ● Screen capture source — Broadcast any application window or full desktop via getDisplayMedia. Preview panel shows the capture live.
  • ● Soundboard — Upload multiple audio files as trigger pads. Audio is mixed into the broadcast stream via WebAudio. Right-click a pad to remove it.
  • ● Soundboard master volume — SFX VOL slider controls the overall gain for all soundboard pads.
  • ● Microphone is now always captured in screen-capture mode for live commentary.
v1.1.0
2 Apr 2026
Feature
Admin Panel & Host-Based Routing
  • ● Admin panel at /admin — Authenticated view listing live channels; supports toast, image, video, and audio push commands.
  • ● Force takedown — Admins can kick any broadcaster off-air. The broadcaster sees an alert; viewers get a "broadcaster left" notice.
  • ● Host-based routing — watch.*, broadcast.*, admin.* CNAMEs all resolve to the same Render deployment; Express rewrites paths by Host header.
  • ● Flash Overlay — Broadcasters can trigger a fullscreen image overlay for a configurable duration.
v1.0.0
18 Mar 2026
Initial
Initial Release
  • ● WebRTC broadcasting — Canvas-captured stream delivered to multiple viewers via peer connections negotiated over Socket.IO.
  • ● Multi-channel support — Multiple broadcasters can operate simultaneously, each on a unique channel ID.
  • ● Playlist playback — Upload multiple video files; plays sequentially with loop, skip, and seek controls.
  • ● WebCam source — Switch between file playback and live camera as broadcast source.
  • ● Graphics layer — Live tag, date/time stamp, on-screen text, and scrolling ticker rendered via canvas.
  • ● Mixer — Mic volume, video audio mute, brightness, contrast, and saturation controls.
  • ● VU meter & LED bar — Real-time audio level visualisation via Web Audio API analyser node.
  • ● TV client — "Anchorage" branded viewer page with channel list, TV bezel UI, volume control, and status bar.
